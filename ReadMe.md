# Иснтрукция по заупску

Для начала клонируйте репозиторий GitHub на свой компьютер

```
git clone https://github.com/sweet-kotik/HackerNews-frontend.git
```

Затем перейдите в каталог проекта и установите зависимости с помощью npm

```
npm i
```

Затем запустите приложение с помощью npm

```
npm run start
```

Приложение будет доступно по адресу http://localhost:3000.  
Для создания билда введите команду

```
npm run build
```
